## 📌 Sobre o Projeto

Este projeto simula a tela de seleção de perfis da Netflix, permitindo visualizar diferentes usuários e interagir com a interface de forma moderna e responsiva.

Além da interface, foram implementadas melhorias de usabilidade e experiência do usuário, como:

* Alternância entre **modo claro 🌞 e escuro 🌙**
* Persistência de tema usando **LocalStorage**
* Interface responsiva (mobile + desktop)
* Estrutura semântica e acessível

## 🚀 Tecnologias Utilizadas

* **HTML5** → Estrutura semântica
* **CSS3** → Estilização moderna com variáveis (CSS Variables)
* **JavaScript (Vanilla)** → Interatividade e manipulação do DOM

---

## 🎨 Funcionalidades

✔️ Seleção de perfis de usuário
✔️ Botão para alternar entre modo claro e escuro
✔️ Tema salvo automaticamente no navegador
✔️ Efeitos visuais (hover, sombra, transições)
✔️ Layout responsivo

---

## 🧠 Boas Práticas Aplicadas

* Separação de responsabilidades (HTML, CSS, JS)
* Uso de **CSS Variables** para facilitar manutenção
* Código JavaScript organizado e reutilizável
* Acessibilidade com `aria-label`
* Uso de `localStorage` para persistência de estado
* Responsividade com `media queries`
